# Portal de IA - Back

A versão do python a ser utilizada e a versão 3.12

## Setup

Para configurar o setup: cria variáveis de ambiente e instala as bibliotecas, basta executar `setup_venv.sh`. Caso ocorra erro de permissão, conceda permissão com `chmod +x setup_venv.sh` e execute novamente.

## Executar

Para executar o projeto, basta rodar `run.sh`. Caso ocorra erro de permissão, conceda permissão com `chmod +x run.sh` e execute novamente.


Criar ambiente virtual
🍏 MacOS & 🐧 Linux (Ubuntu/Debian/Fedora/etc.)
```bash
python3 -m venv .venv && source .venv/bin/activate && pip install -r requirements.txt
```


🪟 Windows
```bash
python3 -m venv .venv && .venv\Scripts\activate && pip install -r requirements.txt
```
